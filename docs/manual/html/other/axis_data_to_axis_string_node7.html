Architecture

The only module is the axis_data_to_axis_string module. It is listed below.

  • axis_data_to_axis_string Implement an algorithm to convert input data to ASCII string (see core for documentation).

The only always process converts the input data to a stream of ASCII strings.

  1. If the counter has data and the desitination device is ready, output data and decrement buffer counter.
  2. If the input data is valid and the counter is 0, meaning previous ASCII string is exhausted, take input data and create ASCII string.
    1. Insert data prefix into buffer.
    2. Using unrolled for loop encode tdata input into ASCII HEX and insert into buffer (4 bit nibbles 0 to F).
    3. Insert delimiter into buffer.
    4. Insert user prefix into buffer.
    5. Using unrolled for loop encode tuser input into ASCII HEX and insert into buffer (4 bit nibbles 0 to F).
    6. Insert delimiter into buffer.
    7. Insert destination prefix into buffer.
    8. Using unrolled for loop encode tdest input into ASCII HEX and insert into buffer (4 bit nibbles 0 to F).
    9. Insert string termination into buffer.
    10. Counter is set to string length.
